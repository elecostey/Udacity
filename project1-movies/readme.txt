My Favorite Movies Project version 1.0.0 13/10/2016

GENERAL USAGE NOTES
-------------------------------------------------------------------------------
- Running the "entertainment_center.py" generates a html file of the website
(My Favorite Movies) based on the provided movie information in
"entertainment_center.py".
- After successful HTML generation, the default OS web browser is opened
displaying the "my_favorite_movies.html" file.

REQUIREMENTS
-------------------------------------------------------------------------------
- This software requires Python Interpreter 2.X.X to run.
- This software was built using Python 2.7.12

INSTALLATION GUIDELINES
-------------------------------------------------------------------------------
- This software does not require any special installation.
